            Follow the below step to Software Installations

Step-1
    - First install frontend and backend Node-Module.

  //---Node Module---//
        npm install

  //---Angular Module---//
        cd .\frontend\
        npm install

Step-2
    - To cerate Admin Account and seed books in database.

   //--- Create Admin and Books---// 
        npm run seeder

step-3
    - Then start the frontend and backend server.

    //---Node Server---//
         npm start

    //---Angular Server---//
        cd .\frontend\
        ng s
